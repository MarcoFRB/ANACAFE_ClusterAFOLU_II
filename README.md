# ANACAFÉ AFOLU II
## Aplicación para el manejo de sombra del cafetal mediante Inteligencia Artificial y Observación de la Tierra

---

## Descripción

Este repositorio contiene el desarrollo de una aplicación para la estimación y manejo de la sombra en sistemas cafetaleros utilizando fotografías hemisféricas, visión por computadora, redes neuronales convolucionales (CNN), imágenes satelitales y herramientas de Google Earth Engine.

El proyecto forma parte de las actividades desarrolladas en el curso AFOLU II del Clúster Copernicus LAC y busca fortalecer la agricultura climáticamente inteligente mediante tecnologías geoespaciales e inteligencia artificial.

---

## Objetivo

Desarrollar una herramienta que permita:

- Estimar automáticamente el porcentaje de sombra del cafetal.
- Clasificar niveles de cobertura arbórea.
- Integrar información satelital y climática para apoyar decisiones de manejo.
- Generar recomendaciones técnicas para mejorar la productividad y resiliencia del cultivo de café.

## Flujo de trabajo

1. Adquisición de fotografías hemisféricas.
2. Preprocesamiento de imágenes.
3. Entrenamiento de la red neuronal convolucional.
4. Validación del modelo.
5. Estimación del porcentaje de sombra.
6. Generación de recomendaciones de manejo.
7. Integración con Google Earth Engine para análisis espacial.
